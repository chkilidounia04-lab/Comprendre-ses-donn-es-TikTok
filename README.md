# Audit Citoyen & RGPD

Ce projet est un outil d'analyse de données personnelles TikTok utilisant le droit à la portabilité (Article 20 du RGPD). 

## Fonctionnalités
- **Recherche Récursive** : Extraction intelligente des données malgré les changements de structure de la plateforme.
- **Analyse de Sentiment** : Classification de plus de 1500 interactions textuelles.
- **Visualisation Temporelle** : Identification des cycles de consommation (addiction numérique).

## Éthique et Confidentialité
- Traitement **100% local** (Privacy by Design).
- Les données personnelles sont exclues via `.gitignore`.
